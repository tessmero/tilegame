Tile-based simulation game with basic map editing and save/load from strings.

Uses [LZString](https://pieroxy.net/blog/pages/lz-string/index.html) to compress and decompress map layouts in json form.

## Demo

https://tessmero.github.io/tilegame.html

## Usage

clone this repository to your computer

open `test.html` to test locally, using source files in `lib` and `src`

run `python build.py` to concatenate all source files into one distributable file: `production.js`
